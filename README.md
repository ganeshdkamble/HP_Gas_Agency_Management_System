# 🛢️ HP Gas Agency Management System

A complete Gas Agency Management System developed using **Python, Flask, MySQL, HTML, CSS, and Bootstrap**. This application helps gas agencies manage customers, bookings, employees, payments, and deliveries efficiently.

---

## 📌 Project Overview

The HP Gas Agency Management System is designed to automate daily operations of a gas agency. It provides an easy-to-use interface for managing customer records, LPG bookings, deliveries, employees, and payment information.

## 💻 Installation

### Clone Repository

```bash
git clone https://github.com/ganeshdkamble/HP_Gas_Agency_Management_System.git
```

### Move to Project Folder

```bash
cd HP_Gas_Agency_Management_System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure MySQL Database

- Create Database
- Import SQL File
- Update database credentials in `config.py`

### Run Application

```bash
python app.py
```
